⚙️ First-Time Setup
Connect to your GitHub repository for data storage. This only needs to be done once — settings are saved in this browser.
How to get a GitHub Token:
1. Go to github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. Click Generate new token (classic)
3. Give it a name like CampAlloc
4. Set expiration to No expiration (or 1 year)
5. Tick repo (full control of private repositories)
6. Click Generate token and copy it here
